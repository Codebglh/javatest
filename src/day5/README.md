#### 2.访问修饰符

    public 对外公开
    
    protected 对子类和同一个包公开
    
    private 只有类本身可以访问，不公开
    
    默认 向同一个包的类公开

    2）、对象的多态
                A.一个对象的编译和运行类型可以不一致

                C.运行类型可以改变
